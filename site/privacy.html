Privacy Policy

Last updated: March 20, 2026

brand.md is an open-source project that provides a file format specification and a Claude Code plugin for generating brand identity files.

Data Collection

The brand.md plugin does not collect, store, or transmit any user data. All processing happens locally within your Claude Code session. We have no servers, no databases, no analytics, and no tracking.

Website

This website is a static site hosted on Cloudflare Pages. It does not use cookies, analytics, or any form of tracking. No personal information is collected when you visit this site.

Third-Party Services

When you use the /brand-md:brand skill, it runs within Claude Code and may use web search and scraping tools (provided by your Claude Code environment) to research your market. This data is processed by Claude and is subject to Anthropic's privacy policy.

Open Source

The complete source code is available at github.com/caiopizzol/brand.md under the MIT License. You can inspect exactly what the plugin does.

Contact

For questions about this policy, open an issue on GitHub.
